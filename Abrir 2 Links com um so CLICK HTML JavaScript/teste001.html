Teste de Abertura de 2 Links ao mesmo tempo

<!--   Código HTML com o atributo JavaScript   --> 

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=p, initial-scale=1.0">
    <title>Abertura de links</title>
</head>
<body>
    <h1>Teste de Abertura de 2 Links ao mesmo tempo</h1>
    <p>
        Clique <a href="pag002.html" target="_blank" onclick=" location.href = 'pag001.html'">
            aqui
        </a> para um teste.
    </p>
</body>
</html>

      A propriedade onclick do JavaScript é um ESCUTADOR DE EVENTO que monitora o atributo tag <a> e mudará para a página em questão e a segunda página será aberta em uma nova aba devido ao atributo '_blank'.

Clique aqui para que o resultado possa ser visualizado.
