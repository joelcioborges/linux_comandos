Como instalar o aplicativo GitHub Desktop no Linux

18 de setembro de 2024 Por Edivaldo Brito
FacebookTweetPinLinkedIn

Se você é usuário do GitHub e quer facilidades ao trabalhar no desktop, conheça e veja como instalar o aplicativo GitHub Desktop no Linux.

GitHub Desktop é um aplicativo GitHub de código aberto. Esta versão do Github Desktop é um fork que adiciona suporte para Linux

O GitHub Desktop é um aplicativo GitHub baseado em Electron de código aberto. Ele foi escrito em TypeScript e usa React.

Como instalar o aplicativo GitHub Desktop no Linux

Para instalar o aplicativo GitHub Desktop no Linux, use um dos procedimentos abaixo.

Como instalar o aplicativo GitHub Desktop no Linux via arquivo AppImage

Para instalar o aplicativo GitHub Desktop no Linux via arquivo AppImage, você deve fazer o seguinte:

Passo 1. Abra um terminal;
Passo 2. Confira se o seu sistema é de 32 bits ou 64 bits, para isso, use o seguinte comando no terminal:

uname -m

Passo 3. Se seu sistema é de 64 bits, use o comando abaixo para baixar o programa. Se o link estiver desatualizado, acesse essa página, baixe a última versão e salve-o com o nome githubdesktop.appimage:

wget https://github.com/shiftkey/desktop/releases/download/release-2.9.0-linux4/GitHubDesktop-linux-2.9.0-linux4.AppImage -O githubdesktop.appimage

Passo 4. Dê permissão de execução para o arquivo baixado;

chmod a+x githubdesktop.appimage

Passo 5. Agora instale o programa com o comando;

sudo ./githubdesktop.appimage --install

Passo 6. Quando você inicia o arquivo pela primeira vez, será perguntado se deseja integrar o programa com o sistema. Clique em “Yes” (Sim) se quiser integrá-lo ou clique em “No” (Não), se não quiser. Se você escolher Sim, o iniciador do programa será adicionado ao menu de aplicativos e ícones de instalação. Se você escolher “No”, sempre terá que iniciá-lo clicando duas vezes no arquivo AppImage.

Como instalar o aplicativo GitHub Desktop via Flatpak

Para instalar o aplicativo GitHub Desktop no Linux via Flatpak você precisa ter o suporte a essa tecnologia instalado em seu sistema. Se não tiver, use o tutorial abaixo para instalar:

Como instalar o suporte a Flatpak no Linux

Depois, você pode instalar o aplicativo GitHub Desktop via Flatpak, fazendo o seguinte:

Passo 1. Abra um terminal;
Passo 2. Use o comando abaixo para instalar o programa via Flatpak. Tenha paciência, pois muitas vezes pode demorar vários minutos para que o Flatpak faça o download de tudo o que precisa;

flatpak install --user https://flathub.org/repo/appstream/io.github.shiftey.Desktop.flatpakref

Passo 3. Para atualizar o programa, quando uma nova versão estiver disponível, execute o comando:

flatpak --user update io.github.shiftey.Desktop

Passo 4. Depois, se fornecessário, desinstale o programa, usando o seguinte comando no terminal:

flatpak --user uninstall io.github.shiftey.Desktop

Ou

flatpak uninstall io.github.shiftey.Desktop

Pronto! Agora, você pode iniciar o programa no menu Aplicativos/Dash/Atividades ou qualquer outro lançador de aplicativos da sua distro, ou digite github ou em um terminal, seguido da tecla TAB.
