Dicas Linux

HD Externo Não é Reconhecido: Ubuntu

Primeira opção:

  1. Abrir o Programa Disc (discos)
  2. Procurar o HD que não está sendo reconhecido e procurar no menu a FORMA DE MONTAGEM do HD
  3. Desligar a forma padrão do usuário
  4. Abaixo do ícone simbólico adicionar: [,ro] (sem os colchetes)

Segunda opção:

  1. Abrir o programa disc (discos) do Linux e procurar a identificação do caminho do HD não reconhecido
  2. Corrigir um provável erro no sistema de arquivos com o comando: 
    sudo ntfsfix "caminho do seu dispositivo"

Terceira opção

  1. Procurar o programa Disk no linux
  2. Escolher a opção Corrigir sistema de arquivos
